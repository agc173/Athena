## 2026-02-18 (slice MVP horóscopo diario)

### Feature
- Se implementa el primer vertical slice real de astrología: Horóscopo diario (mock determinista por signo).
- Domain: ZodiacSign, DailyHoroscope, HoroscopeRepository, GetDailyHoroscopeUseCase.
- Data: HoroscopeRepositoryImpl sin red, con fecha fija `2026-02-18` para comportamiento determinista.
- Presentation: HoroscopeUiState + HoroscopeViewModel (StateFlow, carga inicial de Aries, refresh y cambio de signo).
- UI Compose: nueva HoroscopeScreen como pantalla principal temporal.

### Tests
- Domain: tests del use case (success + propagación de signo).
- Presentation: tests de ViewModel (init, selección de signo, manejo de error).

Impacto:
- Sin cambios de módulos, Gradle o contratos API.
- Se valida el flujo Domain → Data → Presentation → UI con DI en Koin.
